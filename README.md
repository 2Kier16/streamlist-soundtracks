A React-based web app that lets users catalog their favorite movie soundtracks and composers. Designed with a clean, accessible layout and intuitive navigation, StreamList is built for music lovers, film buffs, and anyone who wants to track the scores that move them.

##  Features

- Responsive navigation bar with routing via React Router
- Styled input form for adding soundtracks or composers
- Clean, dark-themed UI with accessible contrast
- Modular component structure for easy expansion

## Technologies Used

- React
- React Router
- CSS Flexbox
- JSX
- Git & GitHub
